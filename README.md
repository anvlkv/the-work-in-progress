## Commands

**Install Dependencies**

```console
yarn
```

**Start TTS server**

```console
docker run -it -p 5500:5500 synesthesiam/opentts:en --cache
```

**Start Preview**

```console
yarn start
```

**Render video**

```console
yarn render <CompositionId>
```

**Upgrade Remotion**

```console
yarn run upgrade
```
